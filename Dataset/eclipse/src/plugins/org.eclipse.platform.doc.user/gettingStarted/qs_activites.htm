Activities Editor

To launch the activities editor:

  1. Download the activities editor zip file, extract the contents into the Eclipse plug-in directory and start Eclipse.
  2. From the main menu bar, select File > New > Activity Binding.

    Notes:

    • To create a new activities.xml file, select File > New > File.
    • Once an activities.xml file has been created, double-clicking it will launch the tool.
    • If the Activity Binding menu item or Eclipse icon is not visible, the activity editor can be added to the perspective by selecting Window > Customize Perspective and checking the Activity Binding command group within the Commands tab.

Supported Operations

Operation Instructions

Add a selected plug-in to a selected activity in the categories, activities, and plug-ins and fragments viewer.

 Click ">>" or select "Add" from the menu accessed by right clicking in the plug-ins and fragments viewer.
Delete a selected category and/or activity in the categories, activities, and plug-ins and fragments viewer. Select "Delete" from the menu accessed by right clicking in the viewer.
Remove selected categories and/or activities, and selected plug-ins from the categories, activities, and plug-ins and fragments viewer. Select "Remove/Delete" from the menu accessed by right clicking in the viewer.
Remove selected plug-ins from the categories, activities, and plug-ins and fragments viewer and return them to the plug-ins and fragments viewer. Click "<<" or select "Remove" from the menu accessed by right clicking in the categories, activities, and plug-ins and fragments viewer.
Create a category in the categories, activities, and plug-ins and fragments viewer. Click "New Category" or select "New Category" from the menu by right clicking in the viewer.
Create an activity in the selected category in the categories, activities, and plug-ins and fragments viewer. Click "New Activity" or select "New Activity" from the menu by right clicking in the viewer.

Edit the description field for the categories and activities.

 Edit the descriptions page as required.
